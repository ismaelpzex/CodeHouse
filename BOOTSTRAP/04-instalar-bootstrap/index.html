Instalación de bootstrap

  • creamos el package.json => npm init
  • Instalamos bootstrap en entorno desarrollo => npm install bootstrap -D
  • Con programas como Purge podremos elminiar las clases que no estamos usando

Instalación de PurgeCSS

  • Se instala de forma global npm install purgecss
  • Se ejecuta => purgecss -css /path/file.css --content /path/file.html --output /path/new.css
